# Weather Dashboard Application

This project is a weather dashboard built with React, integrating data from the OpenWeatherMap API.

## Setup

### Prerequisites

Before running this application, make sure you have the following installed:

- Node.js (https://nodejs.org/)
- npm or yarn (https://www.npmjs.com/ or https://yarnpkg.com/)

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd <repository-folder>

### API Key
Obtain API Key from OpenWeatherMap
To fetch weather data, you need an API key from OpenWeatherMap. Follow these steps:

Sign up for a free account on OpenWeatherMap: https://home.openweathermap.org/users/sign_up

After signing in, navigate to the API Keys section to generate a new API key.

Create a file named .env in the root of the project.

Add your API key to the .env file:
